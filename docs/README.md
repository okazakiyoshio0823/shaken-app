## システム概要

### システム名

**Mechaniq（メカニーク）** - 車検見積管理システム

### システム概要図

```
┌──────────────────────────────────────────────────────────────┐
│                      クライアント                             │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │  フロントエンド（HTML + Vanilla JavaScript）             │ │
│  │  - index.html（メイン）                                  │ │
│  │  - login.html（認証）                                    │ │
│  └─────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────┘
                              │
                              │ REST API (HTTP/JSON)
                              ▼
┌──────────────────────────────────────────────────────────────┐
│                        サーバー                               │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │  バックエンド（Node.js + Express）                       │ │
│  │  - 認証・セキュリティ                                    │ │
│  │  - 顧客・見積管理API                                     │ │
│  └─────────────────────────────────────────────────────────┘ │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │  データベース（SQLite/PostgreSQL）                       │ │
│  │  - ユーザー・顧客・見積データ                            │ │
│  └─────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────┘
```

### 主要機能

- ユーザー認証（2FA対応）
- 顧客・車両情報管理
- 整備見積作成
- 法定費用自動計算
- PDF出力（見積書/請求書/領収書）
- データ保存・履歴管理

## クイックスタート

### 1. サーバー起動

```bash
cd server
npm install
npm start
```

### 2. アプリケーション起動

ブラウザで `index.html` を開く

### 3. 初期ログイン

- ユーザー名: `admin`
- パスワード: `admin`
